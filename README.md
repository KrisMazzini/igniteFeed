## ℹ️ About

This project consists of a new social media for developers: the ignite feed.

In this page users should be able to create posts about their own projects, interact
with other users throught feedback comments and so on.

## 💻 Clone

1. clone this repository using ```git clone git@github.com:KrisMazzini/igniteFeed.git``` ;
2. open the console at the project directory;
3. install the dependencies using ```npm i``` or ```npm install```;
4. execute ```npm run dev``` in the command line;
5. open <http://localhost:3000> on your browser;
